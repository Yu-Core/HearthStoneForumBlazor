# HearthStoneForumBlazor

#### 介绍
炉石传说论坛的Blazor重制版，使用Blazor和ASP.NET Core WebApi前后端分离构建,具有响应式显示。

#### 前端

- 主要语言：C#
- 主要技术栈：Blazor + Masa Blazor + Blazored.LocalStorage
- .NET版本：.NET6

#### 后端

- 主要语言：C#
- 主要技术栈：ASP.NET Core WebApi + SqlSugar + JWT + AutoMapper
- 数据库：SQL Server
- .NET版本：.NET6
