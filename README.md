# HearthStoneForumBlazor

#### 介绍
炉石传说论坛的Blazor重制版，使用Blazor和ASP.NET Core WebApi前后端分离构建。

#### 信息

- 语言：C#
- 框架：Blazor + ASP.NET Core WebApi
- 版本：.NET6
